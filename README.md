# Weather Website

A simple and responsive weather web app that fetches real-time weather data using a public API.

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Getting Started
```bash
git clone https://github.com/mrsandeep27/WEATHER-WEBSITE.git
cd WEATHER-WEBSITE
```

Open `index.html` in your browser.

> **Note**: Add your API key from [OpenWeatherMap](https://openweathermap.org/api) to `main.js`.

## Project Structure
```
index.html      # Main page
contactus.html  # Contact page
main.js         # Weather API logic
main.css        # Styling
bg.jpg          # Background image
contact.jpg     # Contact page image
```

## Features
- Real-time weather data via API
- Responsive layout for all screen sizes
- Contact page
